# ConnectHub

A social media web application built with Django.

## Features

- User Registration & Login
- Create Posts
- Upload Images
- Like & Unlike Posts
- Comment on Posts
- User Profiles
- Follow & Unfollow Users
- Edit Profile
- Profile Pictures

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS

## Installation

```bash
git clone https://github.com/shaik786afsa-star/ConnectHub.git
cd ConnectHub
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
